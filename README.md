# OpenRocket Dispersion Analysis

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)]()
[![Code Style](https://img.shields.io/badge/code%20style-black-000000.svg)]()

A Python-based toolset developed for the Waterloo Rocketry design team to parse, analyze, and visualize Monte Carlo flight simulation data generated by Waterloo Rocketry's [or-monte-carlo plugin](https://github.com/waterloo-rocketry/or-monte-carlo). 

This repository provides a streamlined data pipeline to evaluate trajectory variations, landing footprints, and critical flight dynamics limits, ensuring launch vehicles meet stringent performance and safety criteria for events like the Launch Canada Challenge.

## Features

- **Automated Data Ingestion:** Rapidly parses bulk `.csv` outputs from OpenRocket Monte Carlo runs.
- **Landing Footprint Mapping:** Plots 2D scatter maps of projected landing coordinates to assess downrange and crossrange drift for recovery operations.

## Tech Stack

The analysis pipeline is built for high-performance scientific computing and relies on the following core libraries:
- `pandas` for efficient loading, cleaning, and handling of large simulation datasets.
- `numpy` and `scipy` for underlying statistical calculations and array manipulations.
- `matplotlib` for generating robust, publication-ready engineering plots.
